# 如何修改问卷内容和应用内文字

你好！这份文档会指引你如何修改本应用中的问题和大部分的界面文字。即便你对前端开发不太了解，也可以按照以下步骤轻松完成修改。

## 1. 修改问卷的问题和Bob的答案

所有的问卷题目、以及Bob为每道题预设的答案都存放在一个独立的文件中，方便你集中修改。

- **文件路径:** `data/questions.json`

这个文件包含一个题目列表。每个题目都是一个对象，结构如下：

```json
{
  "id": 1,
  "type": "text",
  "question": "你最喜欢的电影是什么？",
  "bobAnswer": "《爱在黎明破晓前》",
  "photoUrl": "https://images.unsplash.com/..."
}
```

### 如何修改:

1.  打开 `data/questions.json` 文件。
2.  **修改问题:** 找到你想要修改的题目，直接编辑 `question` 字段后面的文字。
3.  **修改Bob的答案:** 直接编辑 `bobAnswer` 字段后面的文字。
4.  **修改题图 (可选):** 如果你想更换某一题的题图，可以替换 `photoUrl` 中的图片链接。

**例如，** 如果想把第1题改成“最想去哪里旅游”，并且Bob的答案是“冰岛”，你可以这样修改：

```json
// 修改前
{
  "id": 1,
  "type": "text",
  "question": "你最喜欢的电影是什么？",
  "bobAnswer": "《爱在黎明破晓前》",
  "photoUrl": "..."
}

// 修改后
{
  "id": 1,
  "type": "text",
  "question": "最想去哪里旅游？",
  "bobAnswer": "冰岛",
  "photoUrl": "..."
}
```

**注意:** 请确保只修改双引号 `""` 内的文字内容，不要删除逗号 `,` 或双引号。

### 如何为题目添加/更换图片

每个问题顶部都可以显示一张图片。我们推荐将图片放在项目文件夹中，这样最稳定。

1.  **找到 `public` 文件夹:** 在项目的根目录，你会看到一个名为 `public` 的文件夹。
2.  **存放图片:** 将你想要使用的图片文件（例如 `my-photo.jpg`）放入 `public` 文件夹。为了方便管理，你可以在 `public` 文件夹内创建一个 `images` 文件夹，然后把图片放进去。
    -   文件路径示例: `public/images/my-photo.jpg`
3.  **更新 `photoUrl`:** 回到 `data/questions.json` 文件，找到你想要修改的题目，然后将 `photoUrl` 字段的值修改为图片的路径。**路径必须以斜杠 `/` 开头**。

**例如，** 假设你有一张名为 `our-trip.jpg` 的照片，并把它放在了 `public/images/` 文件夹下。你可以这样更新 `photoUrl`：

```json
{
  "id": 2,
  "type": "text",
  "question": "我们最近一次旅行，你最难忘的瞬间是什么？",
  "bobAnswer": "是在海边看日落的时候。",
  "photoUrl": "/images/our-trip.jpg"
}
```

如果你不想显示图片，只需将 `photoUrl` 的值设置为空字符串 `""` 即可：

```json
{
  "id": 3,
  "type": "text",
  "question": "下一个纪念日，你希望怎么过？",
  "bobAnswer": "只要和你在一起，怎样都好。",
  "photoUrl": ""
}
```

## 2. 修改应用内的界面文字

应用内的一些固定文字（例如标题、按钮文字等）分散在不同的页面文件中。以下是主要文字的所在位置。

### 欢迎页面

- **文件路径:** `app/quiz/[id]/page.tsx`

你可以在这个文件中找到并修改以下文字：

- `"嗨，Lulu！"`
- `"Bob 为你准备了一份特别的问卷 ✨"`
- `"一共有 {totalQuestions} 道题目"`
- `"回答每个问题后，你就能看到 Bob 的答案 💭"`
- `"开始答题 →"`
- `"准备好了解 Bob 有多了解你了吗？ 😊"`

### 答题页面

- **文件路径:** `app/quiz/[id]/questions/page.tsx`

你可以在这个文件中修改：

- **输入框提示:** `"请输入你的答案..."`
- **按钮文字:** `"提交答案"`
- **答案一致时的提示:** `"答案一致！你们真默契！"`
- **答案不一致时的提示:** `"没关系，继续了解彼此吧 ❤️"`
- **自动跳转提示:** `"正在跳转到下一题..."`

### 提问页面 (答题结束后)

- **文件路径:** `app/quiz/[id]/ask-bob/page.tsx`

你可以在这个文件中修改：

- **完成标题:** `"恭喜完成所有题目！"`
- **副标题:** `"现在轮到你来提问了 ✨"`
- **提示卡片文字:** `"给 Bob 提问"`, `"想问 Bob 什么问题呢？..."`
- **添加新问题部分的文字:** `"添加新问题"`, `"例如：你最喜欢我的哪一点？"`
- **按钮文字:** `"添加问题"`, `"完成并查看结果"`, `"跳过并查看结果"`

### 结果回顾页面

- **文件路径:** `app/quiz/[id]/results/page.tsx`

你可以在这个文件中修改：

- **页面标题:** `"答题回顾"`
- **副标题:** `"看看你和 Bob 的答案吧！"`
- **答案标签:** `"Lulu 的答案"`, `"Bob 的答案"`
- **Lulu提问部分的标题:** `"给 Bob 的提问"`

### 修改方法

1.  根据上面的路径找到对应的文件。
2.  在文件中搜索你想要修改的文字。
3.  直接在代码中修改双引号或标签内的文本内容。

例如，在 `app/quiz/[id]/page.tsx` 文件中，你会看到这样的代码：

```jsx
<h1 className="text-4xl font-bold text-[#2D2D2D]">
  嗨，Lulu！
</h1>
```

你只需要把 `嗨，Lulu！` 修改成你想要的任何文字，例如 `Hello, Lulu!`，然后保存文件即可。在开发模式下，网页会自动刷新显示你的修改。
